REQUIRED IMAGES FOR SEA WATER ROBOT PROJECT
==========================================

Please add the following images to this img/ directory:

1. robot_project_overview.png
   - Overview image of the autonomous sea water robot
   - Should show the complete robot system
   - Recommended size: 1200x800 pixels or similar aspect ratio

2. robot_project_software_GUI.png
   - Screenshot of the software control interface/GUI
   - Should show the monitoring and control dashboard
   - Recommended size: 1920x1080 pixels or similar

After adding these images, the Sea Water Robot research page at:
http://localhost:3000/research/seawater-robot
will display them properly.

Current robot project features:
- IoT-enabled system
- Raspberry Pi control
- YOLO object detection for debris and algae
- GPS module for autonomous navigation
- Real-time monitoring GUI
- Mission planning capabilities
